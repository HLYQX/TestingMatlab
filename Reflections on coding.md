# 1 心得

## 1.1 关于`A(:)`和`reshape()`

在MATLAB中`A(:)`和`reshape()`是按列优先的原则进行按元素迭代的，如

```
>> A

A =

    0.2348    0.3532    0.8212
    0.0154    0.0430    0.1690
    0.6491    0.7317    0.6477

>> A(:)

ans =

    0.2348
    0.0154
    0.6491
    0.3532
    0.0430
    0.7317
    0.8212
    0.1690
    0.6477

>> reshape(A,[9 1])

ans =

    0.2348
    0.0154
    0.6491
    0.3532
    0.0430
    0.7317
    0.8212
    0.1690
    0.6477
```
